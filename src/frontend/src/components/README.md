# components

Shared and feature UI components, one directory per component.

- `core/`: shared UI primitives (Button, Input, Modal, ...).
- `forms/`: form components (LeaveForm, SignInForm, ...).
- `layout/`: page shells and structural components (BaseLayout, ...).
- `calendar/`: calendar components (CalendarGrid, EmployeeLeaveChip, ...).
- `leave/`: leave management components (MyLeaveTable, LeaveTypeBadge, ...).
- `admin/`: admin-only components (AdminLeaveTable, ...).
